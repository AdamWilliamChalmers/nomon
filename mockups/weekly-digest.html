Design notes
• The digest front page carries one emphasised number (the win); everything else is a quiet pill — this is the "40% more scannable" move.

• Serif display type (Newsreader) on the headline, prompt and share card gives the moment an editorial, considered weight that plain UI sans can't. It's the one typographic flourish.

• The profile axis replaces prose with a single learning→hands-on meter per tool — dense but glanceable.

• Colour stays mark-only, except the share card, where richer gradient is appropriate for a social artifact.
