Deep dive · In the conversation

How Nomon speaks up — quietly, then only when it matters.

A ladder of escalation: a passive strip under a reply, an optional disclosure badge, a reflection prompt before a big hand-off, and — only in Guard mode — a brief, always-bypassable hold. Mirror, never nanny.

A — Signal strip (Ambient / Active)

The lightest touch: a strip that teaches on hover

Monochrome; the name carries the signal. Hover for the principle in plain language, cited against what you actually did. An optional one-line "why" can sit beneath.

D — Guard hold (opt-in)

The firmest touch — and still bypassable

Only in Guard mode, and only on a prompt that clearly conflicts with a goal you wrote. A slightly firmer frame signals a goal is at stake; "Send anyway" is always right there.

The escalation ladder, by mode. Ambient = strips only · Active = strips + reflection prompts + badges · Guard = adds the opt-in send-hold · Ghost = nothing in-session, digest only. The design gets firmer only as the user opts in — never louder by default.
